# 배포 전 최종 체크리스트

## 1. 깃허브 올리기 전
- 기존 배포용 폴더를 완전히 비운 뒤 이 수정본만 업로드
- `.env` 실파일은 올리지 말고 `.env.example`만 참고
- `render.yaml`, `Dockerfile`, `requirements.txt`, `start.sh` 존재 확인

## 2. Render 배포 전
- Start Command가 따로 있다면 `uvicorn app.main:app --host 0.0.0.0 --port $PORT` 와 동일한지 확인
- 디스크 마운트 경로가 `/data` 인지 확인
- 환경변수에 `APPDATA=/data` 설정 확인

## 3. 푸시 테스트 전
- 반드시 `https://` 주소에서 접속
- `192.168.x.x:8000` 같은 내부망 주소에서는 테스트하지 않기
- 아이폰은 사파리에서 홈 화면에 추가 후 테스트
- 안드로이드는 크롬 최신 버전 권장

## 4. 배포 후 첫 점검
- 홈 접속
- 로그인
- 알림 설정 버튼 클릭
- 관리자에서 테스트 푸시 1회 발송
- 모바일 잠금화면/상단 알림 확인

## 5. 문제가 있으면 먼저 볼 것
- 주소가 HTTPS인지
- 브라우저 사이트 권한이 허용인지
- 서비스워커가 새 버전으로 갱신됐는지
- Render 재배포가 완료됐는지
