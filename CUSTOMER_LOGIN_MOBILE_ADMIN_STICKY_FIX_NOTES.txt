수정 내용
1) 고객 로그인/회원가입 이메일 정규화 추가
- 회원가입: 이메일 저장 전 trim + lower 적용
- 고객 로그인: trim + lower 기준 조회 적용
- 관리자 로그인도 동일 normalize 함수 사용
- DB 시작 시 기존 users.email 값도 자동 trim/lower 보정

2) 모바일 입력 보정
- login/signup/admin_login 이메일 input에 autocapitalize=none, autocorrect=off 등 추가
- 모바일 자동 대문자/자동수정으로 인한 로그인 실패 완화

3) 모바일 관리자 상단 빠른 메뉴 고정
- admin_dashboard 기존 sticky toolbar를 모바일에서도 숨기지 않도록 수정
- 모바일에서 가로 스크롤 가능한 고정 퀵바로 보이게 CSS 수정
- admin_user_detail / admin_change_password / admin_login 에도 상단 고정 빠른 메뉴 추가
