변경 사항 요약

1) 회원가입 안내 문구 강화
- 회원가입 시 서비스 알림은 사이트 내부에서 먼저 받을 수 있다고 명시
- 로그인 후 브라우저 푸시 연결 페이지로 자동 이동하도록 흐름 변경

2) 회원가입 직후 내부 알림 자동 생성
- 신규 가입자에게만 보이는 직접 알림 생성
- 브라우저 푸시가 아직 연결되지 않아도 사이트 안에서는 알림을 먼저 받을 수 있도록 보강

3) 로그인 직후 자동 푸시 연결 흐름
- 아직 활성 푸시 구독이 없는 계정은 로그인 직후 /push-setup?auto=1 로 자동 이동
- 고객/관리자 모두 적용

4) 푸시 설정 페이지 개선
- 자동 진입 시 0.9초 후 알림 연결 자동 시도
- 이전 화면/관리자 화면으로 돌아가기 버튼 추가
- 안내 문구를 현실적으로 정리

5) 관리자 페이지 개선
- 관리자 푸시 운영 센터에 '이 브라우저 알림 연결' 버튼 추가
- 관리자도 현재 브라우저를 수신 기기로 쉽게 연결 가능

중요
- 브라우저 푸시는 브라우저/OS 권한 정책상 100% 강제 허용은 불가
- 대신 이번 수정으로 '회원가입 즉시 사이트 내 알림 제공 + 로그인 직후 브라우저 푸시 자동 연결 시도' 구조로 바꿈
